Livro Caixa

O Livro Caixa é um sistema simples para registrar movimentações financeiras de entrada e saída, permitindo o controle básico de fluxo de caixa para uso pessoal.

Para executar a aplicação, clone o projeto, compile com Maven e depois execute o arquivo Livro Caixa.cmd na pasta base ou simplesmente abra o navegador no endereço http://localhost:8080/.

Para alterar o usuário, verifique o arquivo SecurityConfig.java dentro de \src\main\java\com\Livro\Caixa\config.
